# Ouro Workbench Product Tour

Ouro Workbench is a native macOS command center for terminal-native agents. It
does not replace Claude Code, Codex, Copilot CLI, shells, or other TUIs; it makes
them durable, inspectable, recoverable, and governable by a selected Ouro boss
agent.

## Investor Pitch

Developer agents are converging on the terminal. The missing product is not
another chat pane; it is an operating room for terminal agents where sessions
survive app death, work is organized by project, and a trusted coordinator can
answer "what is going on?" without the human spelunking through panes.

Ouro Workbench is that operating room:

- a native Mac terminal wrapper with persistent `screen`-backed sessions
- cmux-style workspaces and terminal sessions for project-scoped work
- first-class CLI detection for Claude Code, OpenAI Codex, GitHub Copilot CLI,
  and arbitrary custom terminal/TUI agents
- an Ouro boss layer that can inspect status, summarize waiting work, and queue
  auditable workspace actions through MCP
- restart-aware recovery that distinguishes still-running, respawnable,
  natively resumable, exited, and manual-action sessions
- release artifacts that are packaged, verified, installable, and CI-gated

## What The Operator Sees

The main window is the workbench: workspaces on the left, boss controls across the
top, the selected terminal in the work area, and compact trust/recovery posture
beside each session. The boss pane can expand for coordination or collapse so
the human can live in the terminal.

Key controls:

- `TTFA` summarizes whether the workbench is ready for hands-off operation.
- `Commands` opens a token-aware command palette for common operator moves,
  diagnostics, Workbench MCP, boss asks, release checks, and selected-terminal
  controls.
- `Watch` toggles background boss monitoring.
- `Check In`, `What's Going On?`, `Waiting On Me?`, `Keep Moving`, and
  `Respond For Me` are direct conversational affordances for the selected Ouro
  boss.
- `Full Screen` turns the selected terminal into the primary surface, with
  Redraw, Ctrl-C, Esc, EOF, and Stop still close at hand.

## Boss-Control Layer

The selected Ouro boss agent sees a structured Workbench prompt through MCP:
workspaces, sessions, CLI identity, trust, executable health, recent action log,
transcripts, recovery plans, and boss-watch posture.

Supported queued actions:

- `launch`, `recover`, `terminate`, `sendInput`
- `createGroup`, `createTerminal`, `moveSession`
- `setTrust`, `setAutoResume`
- `archive`, `restore`

Actions are durable queue entries, not hidden side effects. The native app
drains them, authorizes them, applies them, and records the result in the action
log. Malformed MCP action payloads fail loudly before queueing.

## Persistence And Recovery

Workbench separates the visible native client from the durable terminal process.
Each terminal maps to a deterministic `screen` session name, so app quit,
force-quit, reinstall, and relaunch can reattach instead of losing the terminal.

Computer restart is handled honestly: processes do not pretend to survive a
reboot. Workbench persists enough state to classify each session as natively
resumable, respawnable from checkpoint context, exited, or requiring manual
action. `Recovery Drill` runs that classification without mutating state.

## Release State

Ready for ad-hoc-signed public preview:

- native app icon generated and enforced in bundle verification
- bundled `screen` persistence backend
- bundled MCP executable
- bundled SwiftTerm resources
- artifact manifest with SHA-256, bundle metadata, and dirty-worktree flag
- rollback-safe local installer
- latest-release installer for GitHub Releases
- local support diagnostics zip with summaries only by default
- GitHub release workflow for ad-hoc-signed preview artifacts
- CI checks for Swift tests, native scenario verifier, app bundle, and release
  support scripts
- branch protection on `main` requiring the protected checks

Deferred:

- Apple Developer ID signing and notarization
- Sparkle-style in-app automatic binary replacement
